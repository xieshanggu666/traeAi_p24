## 修复1：后端启动错误 - CommonJS 语法不支持

### 问题描述

后端服务启动失败，报错信息：
```
const { PRICE as AVATAR_FRAME_PRICE } = require('../config/migrateAvatarAndChatSkins');
Unexpected identifier 'as'
```

### 问题原因

CommonJS 模块系统的 `require()` 解构赋值不支持 ES6 模块的 `as` 重命名语法。

### 修复方案

**文件**：`server/routes/shop.js`

修改前：
```javascript
const { PRICE as AVATAR_FRAME_PRICE } = require('../config/migrateAvatarAndChatSkins');
```

修改后：
```javascript
const migrateAvatarAndChatSkins = require('../config/migrateAvatarAndChatSkins');
const AVATAR_FRAME_PRICE = migrateAvatarAndChatSkins.PRICE;
```

## 修复2：聊天框皮肤影响时间戳样式

### 问题描述

聊天框皮肤的 `:style` 绑定应用在整个 `.message-bubble` 容器上，导致 `background` 和 `color` 属性影响了内部的时间戳 `.message-time` 元素，使时间戳背景和颜色被皮肤覆盖。

### 修复方案

**文件**：`client/src/views/Chat.vue`

1. 将皮肤样式从 `.message-bubble` 容器移至 `.message-content` 内容元素上
2. 将 `selfBubbleStyle` / `otherBubbleStyle` 计算属性替换为 `getSelfMsgSkinStyle()` / `getOtherMsgSkinStyle()` 方法
3. 模板中只有 `.message-content` 使用 `:style` 绑定皮肤样式，`.message-time` 不受影响

修改前（对方消息）：
```html
<div class="message-bubble" :style="otherBubbleStyle">
  <div class="message-content">{{ msg.content }}</div>
  <div class="message-time">{{ formatTime(msg.created_at) }}</div>
</div>
```

修改后（对方消息）：
```html
<div class="message-bubble">
  <div class="message-content" :style="getOtherMsgSkinStyle()">{{ msg.content }}</div>
  <div class="message-time">{{ formatTime(msg.created_at) }}</div>
</div>
```

## 修复3：头像框和聊天皮肤对方用户可见

### 问题描述

头像框和聊天框皮肤只有使用者自己能看到，对方用户在聊天页面无法看到。

### 修复方案

#### 后端：在用户资料接口返回头像框和聊天皮肤

**文件**：`server/routes/user.js`

在 `GET /user/profile/:userId` 接口中，新增返回 `avatarFrame` 和 `chatSkin` 字段：

```javascript
const { getUserActiveAvatarFrame, getUserActiveChatSkin } = require('./shop');
const avatarFrame = await getUserActiveAvatarFrame(userId);
const chatSkinInfo = await getUserActiveChatSkin(userId);
user.avatarFrame = avatarFrame;
user.chatSkin = chatSkinInfo;
```

> 注：使用延迟 require 避免与 shop.js 的循环依赖问题

#### 前端：聊天页面获取并展示对方用户的皮肤

**文件**：`client/src/views/Chat.vue`

1. 新增 `otherAvatarFrame` 和 `otherChatSkin` 数据变量
2. 新增 `fetchOtherUserSkin()` 方法，通过 `getUserProfile` 获取对方头像框和聊天皮肤
3. 对方头像的 `AvatarDisplay` 传入 `:avatarFrame="otherAvatarFrame"`
4. 对方消息的 `.message-content` 使用 `:style="getOtherMsgSkinStyle()"`
5. `getOtherMsgSkinStyle()` 从 `otherChatSkin` 读取对方皮肤样式
6. 用户资料弹窗中的头像也展示头像框 `:avatarFrame="profileUser.avatarFrame"`
