# 后端启动错误修复记录

## 问题描述

后端服务启动失败，报错信息：
```
const { PRICE as AVATAR_FRAME_PRICE } = require('../config/migrateAvatarAndChatSkins');
Unexpected identifier 'as'
```

## 问题原因

CommonJS 模块系统的 `require()` 解构赋值不支持 ES6 模块的 `as` 重命名语法。

- **ES6 模块**：支持 `import { PRICE as AVATAR_FRAME_PRICE } from '...'`
- **CommonJS 模块**：不支持 `const { PRICE as AVATAR_FRAME_PRICE } = require('...')`

## 修复方案

### 修改文件
`server/routes/shop.js`

### 修改前
```javascript
const { PRICE as AVATAR_FRAME_PRICE } = require('../config/migrateAvatarAndChatSkins');
```

### 修改后
```javascript
const migrateAvatarAndChatSkins = require('../config/migrateAvatarAndChatSkins');
const AVATAR_FRAME_PRICE = migrateAvatarAndChatSkins.PRICE;
```
